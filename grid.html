➤การใช้งาน Grid ใน Bootstrap

BY Natnicha Feb 10, 2017.

" Bootstrap Grid System " คือ ตอบสนองและคอลัมน์ที่จะจัดอีกครั้งขึ้นอยู่กับขนาดหน้าจอ บนหน้าจอขนาดใหญ่ ก็อาจจะดูดีขึ้นกับเนื้อหาที่จัดในคอลัมน์ที่สาม

แต่บนหน้าจอเล็ก ๆ มันจะดีกว่าถ้ารายการเนื้อหาที่ถูกซ้อน ด้านบนของแต่ละอื่น ๆ


➤ระบบ Bootstrap ตารางมี 4 ชั้น :

XS (สำหรับโทรศัพท์)

SM (สำหรับแท็บเล็ต)

MD (สำหรับเดสก์ท็)

LG (สำหรับเดสก์ท็ขนาดใหญ่)

ชั้นดังกล่าวข้างต้น สามารถรวมกันเพื่อสร้างรูปแบบไดนามิกมากขึ้นและมีความยืดหยุ่น
