➤การใช้งาน Form

BY Natnicha Feb 10, 2017.

องค์ประกอบของ Form ใน HTML นั้น อาจจะประกอบไปด้วย ช่องกรอบช้อความ ปุ่มต่าง ๆ ตัวเลือกต่าง ๆ เพื่อใช้ในการรับข้อมูลจากผู้ใช้ ก่อนที่เราจะศึกษาเรื่ององค์ประกอบของฟอร์มนั้น เรามาดู Syntax ของ Form กันก่อนดีกว่าครับ form ใน HTML นั้น จะขึ้นต้นแท็กด้วย < form > และจะปิดท้ายด้วยแท็ก < /form >

การสร้างตัวเลือกด้วย select

คือ รูปแบบฟอร์มหลายตัวเลือก (< select > คือตัวให้เลือกลงมาหลายข้อ : < option value="ลำดับแทนชื่อของตัวเลือก ) เช่น
